# Final Frontier Launcher

The official launcher for **The Final Frontier Testing**.

## Download

[Download the latest release](https://github.com/buskeit/SS14.Launcher/releases/latest)

Windows users should download `FinalFrontierLauncher_Windows.zip`, extract it, and run
`Final Frontier Launcher.exe`.

Do not run the executable directly from inside the ZIP. The other files in the archive are required by the launcher.
